# Lumio
Lumio is a calm, AI-powered journaling and reflection app 

What is Lumio?

Lumio is an AI-assisted mental health journaling app that helps users slow down, reflect, and gain clarity through writing.

Instead of focusing on streaks or pressure, Lumio emphasizes:

gentle reflection

emotional awareness

daily planning in one calm space

Lumio uses AI as a supportive reflection tool, not a replacement for therapy.
